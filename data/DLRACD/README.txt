    ==================================================
           DLR-ACD: DLR's Aerial Crowd Dataset
    ==================================================
This dataset contains 33 large RGB aerial images with 226,291 person annotation.
The dataset is split into 16 train and 14 test images.
This package contains the annotation of the train set. In order to evaluate your results, please send us the predicted crowd density maps or person detections.
For any detail informaiton about the dataset, please refer to our paper!

Reza Bahmanyar, Elenora Vig, and Peter Reinartz, "MRCNet: Crowd Counting and Density Map Estimation in Aerial and Ground Imagery", BMVC Workshop on Object Detection and Recognition for Security Screening (ODRSS), 2019.
